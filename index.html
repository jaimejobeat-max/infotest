Executive Summary

12 Branches.
One Dashboard.
Real-time Integration.

Layer Studios의 12개 지점에 분산되어 있던 아날로그 스케줄 데이터를 Google Antigravity 기술로 통합했습니다.

데이터 전처리부터 웹 배포까지 전 과정을 자동화하여, 수작업 없는 완벽한 디지털 운영 환경을 구축했습니다.

02

System Mechanism

01. Collection

AI 에이전트가 12개 지점 현황판에 동시 접속, 변경사항을 5분 단위로 감지하여 수집 (90% 시간 단축)

02. Processing

비정형 텍스트 데이터를 LLM을 통해 분석, 지점/시간/파트 정보가 담긴 표준 JSON으로 구조화

03. Visualization

Next.js 대시보드에 즉시 반영되어 전 지점 인력 현황을 하나의 캘린더에서 직관적으로 파악
