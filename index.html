Executive Summary

12개 지점의 분산된 스케줄 데이터를 실시간으로 통합하여 하나의 대시보드로 시각화했습니다.

Google Antigravity 기술을 활용하여 데이터 수집부터 배포까지 전 과정을 자동화함으로써, Layer Studios만의 디지털 운영 환경을 구축했습니다.
02

Mechanism

Collection

AI 에이전트가 12개 지점 현황판에 동시 접속하여 데이터를 수집, 확인 시간을 90% 단축했습니다.

Processing

비정형 텍스트(지점, 파트, 시간)를 분석하여 표준화된 JSON 데이터로 구조화합니다.

Visualization

'마스터 캘린더'를 통해 전 지점 인력 현황을 직관적으로 파악할 수 있도록 시각화합니다.
