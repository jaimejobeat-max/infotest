01 / Summary

Executive Summary

본 리포트는 Layer Studios의 12개 지점에 분산된 아날로그적 스케줄 관리 방식을 Google Antigravity AI 기술을 통해 완전 자동화된 디지털 환경으로 전환한 성과를 다룹니다. 실시간 데이터 동기화를 통해 관리 효율성을 극대화하고 인적 오류를 원천 차단했습니다.

02 / Architecture

System Mechanism

Step 1

Collection

지점별 게시판에 AI 에이전트가 5분 단위로 접속하여 변경 사항을 실시간 감지 및 수집합니다.

Step 2

Processing

비정형 텍스트(지점/시간/내용)를 LLM이 분석하여 표준화된 JSON 데이터 스키마로 변환합니다.

Step 3

Visualization

Next.js 기반의 마스터 캘린더 대시보드에 즉시 반영하여 전 지점 현황을 시각화합니다.

03 / Interface

User Experience

Desktop Dashboard

관리자 전용 대시보드는 넓은 화면을 활용하여 12개 지점의 스케줄을 히트맵 형태로 한눈에 조망할 수 있습니다.

Mobile PWA

현장 매니저를 위한 모바일 뷰는 PWA로 구축되어 앱과 동일한 사용성을 제공하며, 스크롤 없는 직관적인 UX를 지향합니다.
